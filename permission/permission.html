How do folder permission work in Assets

Assets will help users conveniently manage and get all materials they need in the board. There are four folders in Assets: Global, Organization, User, and Board.

We will show you more details of how each folder permission works below.

  • Global
  • Organization
  • User
  • Board

Global

Global only allows the owner of Haptic to add files or folders. It is accessible from any board.

Organization

Organization only allows orgainzation administrator to add files or folders. It is accessible from any board.

User

User only allows current users to add files or folders. It is accessible from any board but only for that user.

Board

Board allows all users to add files or folders. It is only accessible in that board.
